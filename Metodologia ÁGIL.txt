04. 
 JavaScript foi criado em 1995 por Brendan Eich, enquanto trabalhava na Netscape. A linguagem surgiu para tornar páginas web mais interativas e dinâmicas. Inicialmente chamada de Mocha, depois LiveScript, recebeu o nome JavaScript para aproveitar a popularidade da linguagem Java na época.Com o passar dos anos, JavaScript evoluiu bastante através do padrão ECMAScript, tornando-se uma das linguagens mais utilizadas no mundo da tecnologia.


O criador do JavaScript foi Brendan Eich.
Ele desenvolveu a linguagem em apenas 10 dias para a empresa Netscape.Seu principal objetivo era permitir que os navegadores executassem ações dinâmicas sem depender apenas de HTML.
